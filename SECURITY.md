# Security Policy

## Supported Versions

This repository contains documentation only.
For security issues with the Sentorix platform,
see below.

## Reporting a Vulnerability

If you discover a security vulnerability in Sentorix,
please do NOT open a public GitHub issue.

**Email:** security@sentorix.io
**Response time:** Within 48 hours

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your suggested fix (optional)

We will acknowledge receipt within 48 hours and
provide a timeline for resolution.

## Scope

In scope:
- Sentorix Gateway API (api.sentorix.io)
- Sentorix Dashboard (app.sentorix.io)
- Authentication and authorisation bypass
- PII detection bypass
- Audit log tampering

Out of scope:
- This documentation repository
- Social engineering attacks
- Denial of service attacks

## Hall of Fame

We will acknowledge security researchers who
responsibly disclose vulnerabilities (with their
permission).
